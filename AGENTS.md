# AGENTS.md

## Role

You are in charge of SSR Svelte 5 app. You are free to use any modern CSS and JS features.

## Code style

- TypeScript strict mode
- Single quotes, no semicolons
- Use functional patterns where possible

## Stack

- Svelte 5 with runes
- Wired-elements
- File-based sv-router

## Styling

- The app style guideline to make a feeling this app is handwritten by children (including all UI elements)
- Use `wired-elements` where possible (they implement the handwritten style)
- When building interfaces, ensure to stay visually consistent wired-elements approach
